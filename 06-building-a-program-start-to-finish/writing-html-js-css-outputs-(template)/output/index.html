Inverted Index

Overview

An inverted index consists of a list of all the unique words that appear in any document, and for each word, a list of the documents in which it appears. Inverted index object takes a JSON array of text objects and creates an index from the array. The index allows a user to search for texts in the array that contain a specified collection of words.

  • How to create an Index
    1. Upload one or more JSON file(s)
    2. Select a file to create it's index
    3. Click 'Create Index' button
  • How to search through file(s)
    1. Select a file to search
    2. Enter your search word(s)
    3. Click on the 'Search' button
