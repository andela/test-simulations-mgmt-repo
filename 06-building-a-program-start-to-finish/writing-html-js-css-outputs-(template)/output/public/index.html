Welcome

You can get straight to using the app as a guest, but if you sign up/in first you'll have the opportunity to save your indexed files to the cloud.

Upload your file(s)

You can upload one or more JSON files that contain your books' data. Make sure your files are properly structured! They should contain an array of object literals each with a title and a text field.

Index your file(s)

When you've uploaded your files, click on the "Create Index" button. Fouille will index your files and add them to the file list. You can click on the "View Index" button to view the index for each file

Search your file(s)

Check the box beside each file to include them in the search.


That's it! Thank you for using Fouille. :)
